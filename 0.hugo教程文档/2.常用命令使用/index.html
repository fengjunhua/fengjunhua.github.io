    2.常用命令使用

    hugo是主要的命令，用来构建我们的网站

    使用方法

    hugo [flags]
hugo [command]
hugo [command] [flags]

    常用的命令主要有:

    new:        
server:
config:   打印网站的配置
convert:  转换网站的content内容到不同的形式
deploy:   部署网站到云环境
env:      打印hugo的版本的环境变量
gen:      
help: 
import:
list:
mod:
version:

    常用的选项参数主要有:

    -b, --baseURL string 
-D, --buildDrafs

    创建一个新的网站 # 

    hugo new site <网站名字>

    生成静态目录 # 

    hugo
