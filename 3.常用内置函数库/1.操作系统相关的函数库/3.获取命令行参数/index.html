    3.获取命令行参数

    获取命令行参数有三种方式

    1.os.args[]

    2.flag

    3.corba

    1. os.args[] #

    2. flag #

    当我们运行go语言程序的时候，flag包实现了命令行参数的解析，当开始使用flag包的时候，需要将

    type Flag struct {
    Name     string // flag在命令行中的名字
    Usage    string // 帮助信息
    Value    Value  // 要设置的值
    DefValue string // 默认值（文本格式），用于使用信息
}

    import "flag"

// 等所有的flag都注册完了以后,需要
flag.Parse()
