    数组的遍历 #

    数组的遍历有两种方法:

    方法一: for循环遍历

    for i:= 0; i< len(a); i++ {
    fmt.Println(i)
    fmt.Println(a[i])
}

    方法二：for … range遍历

    for index,value := range a {
    fmt.Println(index)
    fmt.Println(value)
}
